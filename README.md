# Openwrt-Psiphon-core
برای اینکه بتوانید فایل اجرایی سایفون (psiphon-core) را خودتان مستقیماً برای معماری پردازنده روترتان یعنی aarch64_cortex-a53 (معماری ۶۴ بیتی ARM) کامپایل و آماده کنید، باید از کد منبع (Source Code) رسمی سایفون استفاده کنید.

هسته سایفون با زبان Go (Golang) نوشته شده است، بنابراین قابلیت کامپایل متقاطع (Cross-Compilation) دارد. این یعنی شما می‌توانید روی کامپیوتر خودتان (ویندوز، مک یا لینوکس) بدون نیاز به سخت‌افزار روتر، خروجی مخصوص روتر را بسازید.

دستورالعمل کامل برای شما آماده کرده‌ام:
# ۱. دریافت کد منبع
git clone https://github.com/Psiphon-Inc/psiphon-v3.git
cd psiphon-v3\psiphon-core

# ۲. دانلود وابستگی‌ها
go mod download

# ۳. تنظیم متغیرهای معماری در پاورشل ویندوز
$env:GOOS="linux"
$env:GOARCH="arm64"

# ۴. کامپایل فایل نهایی
go build -ldflags="-s -w" -o psiphon-core .
